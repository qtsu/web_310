선택자의 종류

html 파일 css 를 통해 꾸밀 수 있고 꾸미기 위해서는 지정된 선택을 해야 한다.

작성법

선택자 { 속성명 : 속성값; }

선택자종류 작성방법 설명
전체선택자 * HTML 내부의 모든태그를 불러서 설정한다.
태그선택자 태그명 HTML 내부의 불려진 모든 태그를 설정
아이디선택자 #아이디이름 속성값 ID의 이름을 불러서 특정ID만 설정
클래스선택자 .클래스명 속성값 class 의 이름을 불러서 특정클래스만 설정
id와 class의 차이점:
id는 한페이지 한번만 사용 중복사용안됨 한번 사용한 아이디이름은 두번사용안됨
class는 중복적용가능, 한 클래스에 여러 클래스명 사용가능 클래스명 한칸띄어쓰기로 여러 클래스명 부여가능
